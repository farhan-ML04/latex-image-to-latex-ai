# 🧠 Image to LaTeX AI (Qwen2-VL + LoRA)

This project fine-tunes a Vision-Language Model (Qwen2-VL) using LoRA to convert mathematical images into LaTeX code.

## 🚀 Features
- Image → LaTeX conversion
- Multimodal LLM (Vision + Language)
- LoRA fine-tuning (efficient training)
- Works on Google Colab (4-bit optimization)

## 🧩 Tech Stack
- Python
- Hugging Face Transformers
- Unsloth (LoRA optimization)
- PyTorch
